# 포켓에듀 랜딩페이지

근로자 안전보건교육 및 5대 법정의무교육을 제공하는 포켓에듀의 공식 랜딩페이지입니다.

## 기능

- 교육기관 인증서 표시
- 10,000개 이상의 콘텐츠 소개
- 담당자 맞춤 서비스 안내
- 협력 기업 소개
- Q&A 섹션
- 구글폼 연동 문의 시스템

## 실행 방법

```bash
pip install -r requirements.txt
python app.py
```

브라우저에서 `http://localhost:5000` 접속

## 배포

Vercel, Railway, PythonAnywhere 등에서 배포 가능

## 기술 스택

- Flask (Python)
- HTML5/CSS3
- JavaScript
- Google Forms 연동
